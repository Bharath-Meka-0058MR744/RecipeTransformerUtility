Transform recipe JSON to new Schema for use by AI tools and Agentic Frameworks

📥 Input Schema

📤 Output Schema

ℹ️ How to Use

  1. Upload a JSON file from your device or paste your recipe JSON in the input area
  2. Click "Try Out Example" to load a sample recipe for testing
  3. Click "Validate" to check if it's already in the correct format (optional)
  4. Click "Transform" to convert to recipe-schema-draft-01.json format
  5. Copy or download the transformed output

📋 Supported Input Formats

  • Complex recipe structures with connectors
  • Simple recipe objects
  • Already formatted recipe-schema-draft-01.json (validation only)

🎯 Output Format

The output conforms to recipe-schema-draft-01.json standard with:

  • Unique recipe IDs (UUID)
  • Structured descriptions (overview & details)
  • Categorized tags and keywords
  • Provenance metadata (status, visibility, timestamps, owner)
  • Application dependencies
  • Compatibility and configuration settings
  • Usage statistics tracking
