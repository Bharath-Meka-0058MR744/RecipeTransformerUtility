ℹ️ How to Use

  1. Paste your recipe JSON in the input area
  2. Click "Validate" to check if it's already in the correct format (optional)
  3. Click "Transform" to convert to newSchema.json format
  4. Copy or download the transformed output

📋 Supported Input Formats

  • Complex recipe structures with connectors
  • Simple connector lists
  • Already formatted newSchema.json (validation only)

🎯 Output Format

The output conforms to newSchema.json standard with:

  • Unique connector IDs
  • Standardized authentication methods
  • Capability definitions
  • Source metadata and configurations
